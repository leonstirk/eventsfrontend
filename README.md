# Event Mapping App

This is a simple web app that fetches events from an AWS API Gateway.

## How to Use
1. Open `index.html` in a browser.
2. Click **Load Events** to fetch event data.
3. Events will be displayed as a list.

## API Configuration
- Update `app.js` to point to your API Gateway.

## Deployment
1. Push this repo to GitHub.
2. Deploy using AWS Amplify or S3 + CloudFront.
